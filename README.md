## 📊 Final Results (IRSTD-1K Dataset)

The model has been evaluated on the **IRSTD-1K infrared small target detection dataset**.

### 🔹 Average Performance

* **PD (Probability of Detection):** 0.5343
* **FA (False Alarm Rate):** 0.010155
